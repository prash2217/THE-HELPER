# The Helper

**Emergency Safety App for Civilians and Officers**
**App Maker:** Shinee

**Description:**
The Helper is a Flutter-based emergency safety app designed to assist civilians and government officers. It allows users to send SOS alerts, report incidents, share real-time locations, and identify criminals using face scanning. The app works offline, uses low power, and ensures critical alerts are queued and sent when connectivity is restored.

---

## Features

* **SOS Button:** Instantly alert trusted contacts and authorities.
* **Emergency Call & SMS:** Sends location automatically.
* **Location Tracking:** Low-power real-time tracking.
* **Offline Support:** Works even with no internet; alerts are queued.
* **Face Scanner:** Identify criminals using front/back camera.
* **Report Incidents:** Users can describe events and attach photos.
* **Officer Dashboard:** Verified officers can monitor alerts on a map.
* **Trusted Contacts:** Add and notify trusted contacts during emergencies.
* **Background Tracking:** App runs efficiently with low battery usage.

---

## Technologies Used

* **Flutter** – Cross-platform mobile framework
* **Firebase** – Authentication, Firestore, Storage, Functions
* **Hive** – Offline storage
* **Geolocator & Map Packages** – Location services & maps
* **Shake Detector** – Trigger SOS by shaking the phone
* **Telephony** – SMS integration
